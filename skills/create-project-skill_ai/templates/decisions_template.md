# {{project_name}} Key Decision Log

## Usage

This file records only high-value decisions. It is not intended to be a running log.
Each decision must answer the following:

- What problem needed to be solved at the time
- What options were considered
- Which option was ultimately chosen
- Why it was chosen
- What trade-offs came with the choice
- How this choice was validated as correct
